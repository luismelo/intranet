===========
ABOUT THEME
=========== 
WP News Stream is a responsive blog style theme, with some handy customization options to make it your own. The primary theme color, background color and image can be easily controlled via the admin.
Upload your own logo, set your social media links and choose whether to display author bios. The home page slider will display posts from the category of your choice. There's also a widget space in the
header so you can easily add your own banner code.

This theme is compatible with Wordpress Version 3.4 and above and it supports the new theme customization API (https://codex.wordpress.org/Theme_Customization_API).

All the image graphics and icons bundled with this theme are created by the author and licensed under the GPL.

Supported browsers: Firefox, Opera, Chrome, Safari and IE9+ (Some css3 styles like shadows, rounder corners and 2D transform are not supported by IE8 and below).

For free themes support, please contact us http://hostmarks.com/contact/
