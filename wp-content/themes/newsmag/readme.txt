== How to build the Homepage ==
Firstly, you should create page, choosing the Homepage Template. And go to Appearance > Widgets > Build Your Homepage. Drag and drop the 
'Newsmag - Homepage Builder' to this area. You can choose style, category, block styles and post number by this widget.


== How to add CONTACT FORM ==
Firstly, you have to install recommended plugin Contact Form 7. In order to add Contact Form, you should go to Contact > Contact Forms , and then you will see the shortcode on the screen. Copy it and paste the contact form template's editor.
